Projects

I went through my GitHub and ooh boy there's a lot of old stuff in there.

OpenJDK RISC-V port. I and several other students tried to port OpenJDK's HotSpot JVM to run on RISC-V on Linux. It was my second year in university and other students were using this project for their thesis.

Pygon. Pygon is a console application designed to make it easier to prepare programming contests. It addresses some gripes I (and many others apparently) have with Polygon. Note, that Pygon isn't related to Polygon in any way.

Invoke. A tool for running submissions in a secure programming contest environment.

CodeHighp. Esoteric programming contests platform. I didn't really intend for it to be open source, there are no docs and code is a bit of a mess (really, just a bit).

Notice a pattern yet?

Gross. A package mananager for Linux From Scratch written in Bash. I was fifteen at the time, give me a break.

UberOS Yeah, this one is weird. Unix-like operating system for ComputerCraft. Virtual File System is named fsd and other artifacts that come with not really knowing how operating systems work. Also it's Lua all the way down.

Smaller projects

I haven't released source code for these, mostly because there's no value in it. Most of it is quick and dirty and was written in a day, never to be used again.

Can't Roll, Can't Control. A puzzle game written in under 48 hours for a game jam. People seem to like it, but voting period is not over yet.

Cutout. A puzzle game written in a week for a game jam. First place in game design, second place in fun and innovation. Among 709 entries. Whew.

@PlayCodenamesBot. Play Codenames with friends! In Russian. Via Telegram. Yeah, in hindsight this is not really practical.
